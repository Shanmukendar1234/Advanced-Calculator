
# Advanced Calculator

This is a responsive and interactive web-based calculator built using **HTML**, **CSS**, and **JavaScript**. The calculator allows you to perform both basic and advanced mathematical operations. 

## Features

- **Basic Operations**: Addition, Subtraction, Multiplication, and Division.
- **Advanced Operations**: Includes advanced functions like:
  - Square root (√)
  - Exponentiation (x², xʸ)
  - Trigonometric functions (sin, cos, tan)
  - Reciprocal (1/x)
  - Hyperbolic cosine (cosh)
  - Exponential function (eˣ)
  - Logarithm (lg)
  - Power of 10 (10ˣ)
  
- **Responsive Design**: Works on all screen sizes and devices.
- **Advanced Operations Dropdown**: A button that opens a dropdown for selecting advanced operations.

## Usage

1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/Shanmukendar1234/advanced-calculator.git
   ```

2. Open the `index.html` file in your browser to start using the calculator.

## Files

- `index.html`: Contains the structure of the calculator.
- `style.css`: Defines the styling for the calculator.
- `script.js`: Manages the functionality and logic of the calculator.
